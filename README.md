# 🎮 Mini Game Hub (Mini Games Website)

This project is a simple and interactive **web-based game collection** built using **HTML, CSS, and JavaScript**.
It contains multiple mini games that users can play directly in the browser.

---

## 🌐 Features

* 🎯 Simple and clean user interface
* ⚡ Interactive games using JavaScript
* 🎮 Multiple games on a single website
* 📱 Beginner-friendly project

---

## 🕹️ Games Included

### 1. 🔢 Guess the Number

* The system generates a random number
* User guesses the number
* Hints:

  * "Too High ⬆️"
  * "Too Low ⬇️"
* Game ends when correct

---

### 2. ✊ Rock Paper Scissors

* Player vs Computer
* Options:

  * Rock ✊
  * Paper ✋
  * Scissors ✌️
* Result:

  * Win ✅
  * Lose ❌
  * Draw 🤝

---

### 3. ❓ Quiz Game

* Total questions: **10**
* Each correct answer = **+1 score**
* Final score shown at the end

---

## 🏠 Home Page

* `index.html` is the main page
* It shows all 3 games
* Navigation to each game
